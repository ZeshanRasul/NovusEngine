# Novus Engine

Novus Engine is a game engine built on top of the Vulkan graphics API. It is designed to be a high-performance, cross-platform engine for creating games and interactive applications.

## Features

- Physically Based Rendering (PBR)
- FXAA (Fast Approximate Anti-Aliasing)
- Bloom
- ACES Tone Mapping
- Cascaded Shadow Maps
- Animation System
- Model Loading (Assimp)
- GLTF2 Support
- Entity Component System (ECS)
